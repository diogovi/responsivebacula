Gestor de Configuración
CFengine es una potente herramienta que le permite definir un conjunto de acciones de administración del sistema que lleva a cabo en el momento necesario, para hacer tareas como crear enlaces y directorio, establecer permisos de fichero, gestionar los procesos en ejecución y eliminar ficheros antiguos. Su sistema de clases le permite usar la misma configuración en múltiples sistemas, con diferentes acciones que se lleven a cabo en función del tipo de sistema, nombre de la máquina o cualquier otro criterio.

La página principal de este módulo lista todas las acciones configuradas en su sistema, las clases a las que aplican, y un resumen de qué hará la acción para cada clase. Puede añadir un nuevo tipo de acción usando el botón bajo la tabla, añadir una nueva clase para una acción existente pulsando el enlace Añadir clase..., o editar una clase existente para una acción pulsando sobre el nombre de la clase en la columna Aplicar a las clases.

Si su configuración actualmente está vacía, debería comenzar por crear una acción Control maestro que determine qué otras acciones se ejecutarán. Cuando añada nuevas acciones, también debe estar seguro de que están listadas en la acción de control maestro. Si no lo están aparecerá No junto a la acción en la columna ¿ Activo ?, y nunca se ejecutará.
