Máquinas de Configuración Remota
Esta página lista las máquinas en que está instalado CFengine, para que pueda activar el procesamiento de acciones configuradas en múltiples máquinas simultáneamente desde un solo origen. Cada una de las máquina debe estar ejecutando el demonio de Gestión de Configuración para que funcione el procesamiento remoto. Dese cuenta de que las máquinas remotas usarán su propia configuración local, no la de este servidor. Para distribuir la configuración maestra a múltiples máquinas, haga que la configuración se copie por sí misma cuando se ejecute usando una acción Copiar ficheros.
