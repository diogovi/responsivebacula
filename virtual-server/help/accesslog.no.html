Webtjener tilgangslogg fil
Dette feltet angir hvilken fil Apache tilgangslogger skrives til for denne virtuelle tjeneren. Hver HTTP forespørsel som besvares av dette nettstedets domene vil bli skrevet til denne filen som en separat linje. Denne bør bare endres dersom du flytter loggingen til en annen katalog som f.eks /var/log/virtualmin eller tjenerens hjemmekatalog.
